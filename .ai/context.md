Simple web application for private use.

It's a helpful tool for managing job offers and recruitment process.

Scrapper fetches new offers from the defined websites, OpenAI API process them and we store them in the database.
Only new offers are stored.

As a user I can:
- define websites to scrape
- fetch new offers
- browse the offers
- filter and sort the offers
- edit the offers
- change the status of the offer / recruitment process
